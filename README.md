# To Do List
This is a python program created for the purpose of creating a list to organize and prioritize tasks.

---

# Requirements
1. Python 3.10 or higher(Recommended)

---

# Functions
a - Add Tasks To The List With Priorities Tied to Each one
r - Remove A Specific Task From The List
c - Clears The Entire List
v - View The To Do List
q - Quits The Program
